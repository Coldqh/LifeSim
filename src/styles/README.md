# LifeSim styles

Глобальный CSS разделён на 20 последовательных частей.

## Правила

- `src/styles.css` — только стабильная точка входа.
- `src/styles/index.css` определяет порядок каскада. Не переставляй импорты без визуального регрессионного теста.
- Редактируй конкретный файл раздела напрямую. Повторно запускать split после обычных правок не нужно.
- Новые крупные системы получают отдельный CSS-файл и импорт в нужной позиции.
- Responsive-переопределения должны находиться после базовых правил, которые они меняют.
- Относительные URL были автоматически скорректированы после переноса из `src/` в `src/styles/parts/`.

## Проверка

`node scripts/refactor-styles.mjs verify`
